# Shift and Add Multiplier

## Overview
This project implements a Shift and Add Multiplier in Verilog. The multiplication process is performed by repeatedly shifting and adding based on the bits of the multiplier, similar to the manual binary multiplication method.

## Features
- Verilog HDL implementation
- Binary multiplication using shift-and-add algorithm
- Efficient and easy-to-understand design
- Includes testbench for functional verification

## Files
- `SHAM.v` – Multiplier design module
- `SHAM_tb.v` – Testbench for simulation
- `constraint.xdc`-Constraint file

## Tools Used
- Verilog HDL
- Xilinx Vivado
